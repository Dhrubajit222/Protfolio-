# 🚀 Personal Portfolio

A modern and responsive developer portfolio built with **React**, **Tailwind CSS**, and **Framer Motion**. This portfolio showcases my journey, projects, technical skills, and passion for building intelligent solutions through **Data Analytics**, **Machine Learning**, and **Artificial Intelligence**.

---

## ✨ Features

* 🎨 Modern & Responsive UI
* ⚡ Smooth Animations with Framer Motion
* 🌌 Interactive Glassmorphism Design
* 💡 Dynamic Mouse Glow Effects
* 📱 Fully Responsive Layout
* 🚀 Optimized Performance
* 🎯 Clean & Minimal Design
* 🌙 Dark Theme Interface
